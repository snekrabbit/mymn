## Dictionary
    { $key: $value, ...}

### Basic Dictionary

    ages = {"lia": 15, "pap": 49}
    hair_color = {"lib": "dark brown", "pap": "old"}

### Nested Dictionary

    people = {
        "lia": {
            "hair_color": "dark brown", 
            "age": 15
        }
    }
